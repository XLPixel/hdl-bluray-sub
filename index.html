Welcome

This repository contains subtitle files for the episodes contained in the Hotel Del Luna Blu-ray boxed set. Hotel Del Luna is a popular South Korean television drama that has captivated audiences worldwide with its compelling story, beautiful cinematography, and memorable performances.

About This Project

This project provides carefully synchronized subtitle files for the Boxed Set, excluding the extras. Whether you're watching for the first time or revisiting this series, these subtitles aim to enhance your viewing experience with well-timed translations.

📝 Note: All subtitle files are synchronized with the Blu-ray discs on a per-disc basis. Programs such as MakeMKV should produce one .MKV file around 33GB to 40GB per 2-episode disc. Playing the file is the equivalent to selecting "view all" option in the Blu-ray menu. Opening credits are only displayed once per disc, with the second episode of the disc playing immediately after the end of the first. All chapter markers should have been preserved by your blu-ray ripping program.

Features

  • High-quality, re-timed subtitles catered for blu-ray backups
  • Translations for all dialogue including episode previews
  • Compatible with major media players

How to Use

To use these subtitles with your Blu-ray player or media player:

  • Download the desired subtitle file from the repository
  • Place the subtitle file in the same directory as your video file
  • Ensure the subtitle filename matches the video filename (same name, different extension)
  • Open the video in your preferred media player
  • Select the subtitle file from the player's subtitle menu if not automatically loaded
